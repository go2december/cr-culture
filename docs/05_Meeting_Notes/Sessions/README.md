# Meeting Sessions

โฟลเดอร์นี้ใช้เก็บบันทึกการประชุมหรือ working sessions รายครั้ง

## Suggested Naming
- `YYYY-MM-DD_<topic>.md`

## Suggested Workflow
1. สร้างไฟล์ใหม่จาก `../../99_Templates/Meeting_Note_Template.md`
2. สรุปบริบท การตัดสินใจ และ action items
3. ถ้ามีข้อสรุประยะยาว ให้เพิ่มรายการใน `../Decision_Log.md`
